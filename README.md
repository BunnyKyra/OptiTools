Description

OptiTools is an optimization tool for Unity, designed specifically for VRChat (SDK3 Worlds) projects.
It improves performance by reducing build size, optimizing textures, and applying smart settings automatically.

Features
- Automatic texture optimization
- Build size reduction
- Performance improvements
- Smart configuration system
- VRChat SDK3 compatible
- Simple Editor interface

Installation (.unitypackage)
Import:
OptiTools.unitypackage

Files will be installed automatically in:
Assets/OptiTools/Editor/
Wait for Unity to compile

Usage
Open:
Tools → Optimizer Tool
Select settings
Run optimization

Requirements
Unity 2022.3.x
VRChat SDK3 (Worlds)
World project

Important
Editor-only tool
Not included in builds
Do not move files outside Editor

Security
The tool is protected against reverse engineering.
